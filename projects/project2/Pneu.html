2.- Data normalization + augmentation

Data augmenation means: Artificially creating new training images by applying small random transformations to existing images.
Instead of collecting more data, we:
Modify existing images slightly
Teach the model to be robust to variations

def load_and_preprocess_image(image_path):
    img = tf.keras.utils.load_img(
        image_path,
        target_size=(224, 224)
    )
    
    img_array = tf.keras.utils.img_to_array(img)
    img_array = img_array / 255.0  # normalization
    
    img_array = np.expand_dims(img_array, axis=0)  # add batch dimension
    return img_array

def load_and_augment_image(image_path):
    img = tf.keras.utils.load_img(image_path, target_size=(224, 224))
    img = tf.keras.utils.img_to_array(img)
    img = img / 255.0

    img = tf.expand_dims(img, axis=0)
    augmented_img = extreme_augmentation(img)

    return img, augmented_img

10.- Conclusion

This project implemented a CNN to classify chest X-ray images as normal or pneumonia. Visually distinguishing between the two classes proved difficult without medical expertise, highlighting the challenge of interpreting medical images. Image normalization and data augmentation were applied to improve generalization, although the results showed that augmentation must be handled carefully, as excessive transformations can distort clinically relevant features and lead to misleading predictions. Overall, the exercise emphasizes the importance of proper preprocessing, evaluation, and domain knowledge in medical image classification.
